# 주효 인생 시뮬레이터
세후 월급·생활비를 기반으로 결혼/주택/대출/은퇴/대공황 시나리오를 한 화면에서 돌려보는 자산 시뮬레이터입니다. 모든 입력 단위는 `만원`, 차트와 복사본은 주로 `억원`으로 표시됩니다.

## 다운로드
- Windows portable exe: [다운로드](https://github.com/ajsj2200/Juhyo-Simulator/releases/tag/v0.1.0)

## 주요 특징
- **수입·지출 기반 모델링**: 본인/비교 대상/배우자의 세후 월급·생활비·초기자산·투자액 스케줄을 입력해 실시간 복리/단리 비교.
- **결혼·주택·대출 시나리오**: 결혼 시점/비용, 배우자 은퇴까지 고려해 합산 자산을 계산하고, 주택 구매 시 원리금균등/원금균등/체증식 상환, 중도상환, 집 가치·대출 잔액을 함께 추적.
- **은퇴 및 인출 전략**: 4% 룰, JEPQ/VOO 배분, 인플레이션 반영, 배당으로 생활비 충당 시점 계산.
- **포트폴리오 커스텀**: VOO/SCHD/BND/현금 등 자산 비중 슬라이더, 프리셋, 예상 수익률·표준편차·변동성 레벨, 자동 리밸런싱/몽테카를로 변동성 밴드 옵션.
- **차트 & 토글**: 로그/선형 스케일, 집 포함/제외, 실질가치 전환, 이벤트 라벨(결혼/대출 완료/은퇴), 대공황 음영 표시.
- **몬테카를로/히스토리**: S&P500 연간 수익률 셔플로 전체 시나리오를 2000회까지 반복해 퍼센타일·분포 차트 제공, 연도별 히스토리컬 수익률 적용 모드 지원.
- **대출 계산기**: 명목/실질 월 상환액 라인 차트, 물가상승률 입력, 대출 유형별 초기 상환액 비교.
- **복사·공유**: 결혼/대출/자산/대공황 요약을 텍스트로 복사해 공유 가능.
- **프리셋/로컬 저장**: 단리/복리 비교 프리셋, 은퇴/주택/결혼 기본 활성화, 로컬스토리지에 입력 상태 저장/불러오기/삭제.

## 시나리오 입력 가이드
- **기본 투자**: 연간 수익률, 월 투자액 성장률, 투자액 변경 스케줄을 설정해 장기 복리 곡선을 확인합니다.
- **결혼/주택**: 결혼 연차, 배우자 정보(월급·생활비·투자액·은퇴 시점), 주택 가격/자기자본/LTV/금리/상환 방식/중도상환, 주택 가격 상승률을 입력합니다.
- **은퇴**: 4% 룰/투자 유지 선택, JEPQ 배당률과 배분 비중, 인플레이션, 은퇴 후 생활비를 조정합니다.
- **위기 시나리오**: 대공황 구간을 음영으로 표시하고 실질가치 모드에서 자산 하락을 시각화합니다.
- **포트폴리오**: 포트폴리오 모드를 켜고 자산 비중·리밸런싱 주기·몬테카를로 변동성 밴드를 설정합니다.

## 설치 및 실행 (소스)
```bash
git clone <repo-url>
cd voo-app
npm install
```

### 웹 개발 서버
```bash
npm run dev
```
→ http://localhost:5173

### 린트
```bash
npm run lint
```

## Electron
- 개발
  ```bash
  npm run dev:electron
  ```
- 빌드 (Windows portable exe)
  ```bash
  npm run build:app
  ```
  - 결과물: `release/` 폴더
  - 아이콘 사용 시 `build/icon.ico` 추가

## 주요 파일
- `src/InvestmentCalculator.jsx`: 메인 UI 및 상태 관리
- `src/utils/calculations.js`: 복리/대출/은퇴/대공황/몬테카를로 계산 로직
- `src/constants/defaults.js`: 기본값·프리셋·결혼/은퇴 설정
- `src/constants/sp500History.js`: S&P500 연간 수익률 데이터
- `src/components/*`: 카드/입력/차트/인사이트/프리셋/포트폴리오 UI 컴포넌트
- `electron/main.js`: Electron 엔트리
